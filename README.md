# MayaKinect


## Overview
This project provides a **Maya C++ UI integrated with Python** for realtime motion capture streaming from Kinect devices into Autodesk Maya. It enables live skeletal data capture and animation mapping inside Maya for prototyping and animation workflows.

## Features
- Realtime motion capture streaming from Kinect 360
- Maya UI built with C++ and Python integration
- Live skeleton visualization inside Maya
- Animation retargeting support (in development)
- Modular pipeline for extending device support

## Current Hardware Support
-  Kinect 360 (USB)
-  Kinect for Xbox One (futer plane development)
-  Future plans: real time character animation in maya

## Installation
1. Clone the repository.
2. Build the C++ plugin for Maya.
3. Load the Python UI inside Maya.
4. Connect and configure your Kinect device.
